# 3000 Words

## 👀 Description

This is an Android application designed to learn a basic set of English words that help you understand most English texts and dialogues.

## ⚙️ Features

The application implements 3 modes of learning words: flashcards, manual spelling practice, and multiple choice tests. In each mode, the words are divided into topics, the progress of learning words in all modes (except tests) is saved so that you can pause at any time, and you can reset the progress in each of the topics at any time.
The application does not need the Internet, the AsyncStorage API is used to store and receive data inside the application itself, the entire set of words is stored in the application itself.

## 🛠 Tech stack

- React Native
- Expo
- NodeJS
- AsyncStorage
