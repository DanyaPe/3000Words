# 3000 Слов

## 👀 Описание

Это Android приложение предназначенное для изучения базового набора английских слов, которые помогают понять большинство английских текстов и диалогов.

## ⚙️ Функциональность

В приложении реализованы 3 режима обучения слов: карточки, практика ручного написания и тесты с вариантами ответов. В каждом режиме слова разбиты на темы, прогресс изучения слов во всех режимах (кроме тестов) сохраняется, чтобы можно было сделать паузу в любой момент, также в любой момент можно сбросить прогресс в каждой из тем.
Приложение не нуждается в интернете, для хранения и получения данных внутри самого приложения используется AsyncStorage API, весь набор слов хранится в самом приложении.

## 🛠 Технологический стек

- React Native
- Expo
- NodeJS
- AsyncStorage
